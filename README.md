# 浅色系动漫可爱风 AI 聊天模拟器

本项目为一个纯前端、零依赖、本地持久化存储的 AI 聊天模拟器。

## 项目结构
- `index.html`: 主入口文件，包含所有的 HTML 结构（模态框、主视图、聊天界面）。
- `css/style.css`: 所有的样式定义，包含各种可爱风格的组件样式以及主题切换和响应式布局。
- `js/store.js`: 本地 `localStorage` 数据存取模块（用户、角色、记录、设置）。
- `js/api.js`: AI 接口调用与上下文拼接逻辑。
- `js/ui.js`: DOM 渲染和视图切换相关的纯 UI 逻辑。
- `js/main.js`: 核心业务逻辑绑定，处理各种事件、收发消息、语音和转账等功能。

## 运行方式
由于项目是纯前端，并且使用了模块化的 `.js` 文件加载（如果不使用 ES Module 的 `<script type="module">` 实际上直接双击打开也可能可以运行，但为了最佳兼容性和安全性，强烈推荐通过本地服务器运行）。

### 推荐使用 VS Code Live Server 运行
1. 打开 VS Code。
2. 安装扩展 **Live Server** (作者: Ritwick Dey)。
3. 在 VS Code 中打开本项目的文件夹。
4. 右键点击 `index.html` 文件，选择 **"Open with Live Server"**（或者点击右下角的 "Go Live" 按钮）。
5. 浏览器将自动打开 `http://127.0.0.1:5500/index.html` 并呈现页面。

## 功能特性
1. **多用户与多角色**: 数据完全隔离。
2. **纯本地存储**: 刷新或关闭浏览器不丢失任何数据。
3. **可爱风格与主题切换**: 提供五种浅色系主题无缝切换。
4. **自定义头像形状**: 圆形、圆角、正方形随意选择。
5. **AI API 接入**: 支持通过全局设置配置任何兼容 OpenAI 格式的接口。
6. **多条回复与自动总结**: 支持拉动条调整回复条数 (1-20)，并且带有明显的绿色自动总结记忆开关。
7. **高级转账模拟**: 带有动态鎏金特效的模拟转账功能，支持双向转账。
8. **语音发送模拟**: 按住录音交互，松开发送并附带转文字功能（仅前端视觉模拟交互，不调用真实麦克风录音上传）。

## 注意事项
- 本应用使用手机端界面比例，电脑端查看时居中并带有圆角。
- 请在全局设置中填写真实可用的 API URL 和 API Key 以开启与 AI 的对话。
- 自动总结功能依赖于您填写的 API 服务，触发条件为：每当用户和角色对话累积增加 10 条新消息时，并且在角色设定里填写了“记忆格式”。
